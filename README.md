# CerbeHub: 微服务套装

CerbeHub 由以下几个部分组成:

* endpoint: 基础的端口服务单元

* generator: 端口服务生成器

* engine: 复杂查询引擎

* gateway: API 网关与管理

CerbeHub 目前还是 Proof-of-Concept 阶段，各个组件尽量碎片化处理，以增强其可变性与扩展性，详细的开发路标请参考 [Roadmap](./Roadmap.md)。
